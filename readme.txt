== Description ==

Simple and easy to use contact form to reply directrly from the WordPress Admin Panel and with the functionality to stores messages/queries into database.

== Installation ==

1. Upload `skylite-src-form.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php echo do_shortcode(['src_form']); ?>` in your templates
